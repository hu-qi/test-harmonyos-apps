# Privacy Policy

`Test HarmonyOS Apps` is an open-source developer plugin for Codex-style workflows.

## Data Handling

- This repository does not operate a hosted service.
- The plugin does not intentionally collect, store, or transmit personal data to the project maintainer.
- During normal usage, the plugin may help a user inspect local files, device UI trees, screenshots, and logs from a HarmonyOS or OpenHarmony device.
- Any device data, screenshots, or logs are handled within the user's own development environment unless the user chooses to share them elsewhere.

## Third-Party Services

- GitHub, GitCode, or other platforms used to host the repository may process data according to their own policies.
- HarmonyOS/OpenHarmony tooling such as `hdc`, `hilog`, and `uitest` is provided by their respective vendors and may have separate terms.

## Contact

For repository-related questions, contact the maintainer through [GitHub](https://github.com/hu-qi).
